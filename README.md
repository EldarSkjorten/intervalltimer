# Intervalltimer

En enkel intervalltimer laget for bruk i nettleser på mobil og nettbrett. Ingen installasjon nødvendig — åpne siden og sett i gang.

**Live:** https://eldarskjorten.github.io/intervalltimer/

## Funksjoner

- Stilles inn med øvelsestid, pausetid og antall runder
- Valgfri nedtelling før start
- Visuell ring som viser gjenværende tid per fase
- Lydvarsler ved overgang mellom øvelse og pause
- 5-sekunders varsling (tikk + blink) før hver overgang
- Husker innstillingene dine mellom besøk (localStorage)
- Holder skjermen våken under trening (Wake Lock API)
- Tilpasset for iPhone, iPad og Android — fungerer i Safari, Chrome og Vivaldi

## Planlagte endringer

- **Egendefinerte lyder:** De syntetiske varslingslydene er midlertidige. Planen er å erstatte dem med innspilte lydfiler (mp3) for et bedre og mer motiverende lyduttrykk.
